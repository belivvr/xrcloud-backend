Authentication

해당 문서의 모든 API의 호출에는 ApiKey를 필요로 합니다.

ApiKey는 XRCLOUD Developers에 로그인 한 후 페이지 우상단의 My page에서 발급 받을 수 있습니다.

Project

개발자의 프로젝트, 플랫폼, 3rd-party app 등이 될 수 있는 XRCLOUD의 Project입니다.

Scene

Scene 생성: get project api의 response json 필드 중 sceneCreationUrl을 사용하여 생성합니다.

Scene 수정: get scene api의 response json 필드 중 sceneModificationUrl을 사용하여 수정합니다.

  • optId 값은 XRCLOUD에서 Scene의 생성 및 수정을 확인하기위한 callback value 입니다. 변형하여 사용하면 오류가 발생할 수 있습니다.

Authentication

All API calls in this document require an ApiKey.

  • You can obtain the ApiKey from the My page in the upper right corner after logging into the XRCLOUD page.

Project

A Project in XRCLOUD can be a developer's project, platform, or a 3rd-party application. It is a unit that manages Scenes and Rooms, with multiple Scenes and Rooms belonging to a specific Project.

  • The webhookUrl can be set when creating or modifying a Project on the XRCLOUD page.

Create a Project

You can create a project using the following format:

RES=$(curl -s -X POST "$XRCLOUD_HOST/api/projects" \
    -H "Authorization: Bearer $ACCESS_TOKEN" \
    -H "Content-Type: multipart/form-data" \
    -F "favicon=@$FILE_DIR/favicon.ico" \
    -F "logo=@$FILE_DIR/logo.png" \
    -F "name=lms" \
    -F "label=lms"
)

This API uses multipart/form-data format to include the favicon, logo, name, and label of the project.

Request Body schema: multipart/form-data
required
favicon
string <binary>

Favicon file for the project.

logo
string <binary>

Logo file for the project.

name

string

Scene

A space for creating web-based 3D virtual spaces, necessary for creating Rooms.

By calling the get-creation-url api, you can access the 'sceneCreationUrl' field to access the editor (Spoke) and create a Scene by clicking the Publish Scene button in the upper right corner. Additionally, by calling the get-scene api, you can access the 'sceneModificationUrl' field to make modifications.

  • The optId value is a callback value used to verify the creation and modification of a Scene in XRCLOUD. Modifying it may cause errors.

Room

A Room is a space composed of a Scene created through the editor (Spoke).

Each Room has a private room URL that changes with each information request and a public room URL with a fixed address. These URLs are also divided into a host URL that grants host permissions and a guest URL that grants guest permissions.

The 'returnUrl' refers to the web address to return to when leaving the room, and it can be set in the Rooms tab of XRCloud.

Authentication

This API uses three types of authentication tokens. The AccessToken, ProjectId and ProjectKey.

  • AccessToken is used when making calls to Project APIs. It can be obtained by first registering an account (create admin), followed by logging into that account (login).

ProjectId and ProjectKey are used simultaneously in almost all other API calls.

  • ProjectId can be obtained via create or get project.

  • ProjectKey can be obtained via issue key.

Please ensure that you have the necessary tokens and include them in the header of your requests as required.

Create an authentication token

Creates and returns a bearer token in JWT format that you can use to authenticate with XRCLOUD Developers APIs.

The returned token is used in the HTTP Authorization header like Authorization: Bearer {TOKEN}.

Request Body schema: application/json
email
required
string

The email of the XRCLOUD Developers account to authenticate with.

password
required
string

The password XRCLOUD Developers account to authenticate with.
